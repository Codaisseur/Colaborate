What is indentation?

In computer programming, an indentation style is a convention governing the indentation of blocks of code to convey program structure.
It is especially used to clarify the link between control flow constructs such as conditions or loops, and code contained within and outside of them.

Why would you want to use it?

Indentation shows the structure of the code in an easy to decipher manner. This way, both creater and external users of your code read more easily through your code. In some coding languages, indentation is actually a part of the coding syntax.
